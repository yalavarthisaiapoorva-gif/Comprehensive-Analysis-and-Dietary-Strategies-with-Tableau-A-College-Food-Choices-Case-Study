# 🍎 College Food Choices Analysis and Dietary Insights using Tableau

## 📌 Project Overview

This project explores the eating habits and dietary preferences of college students using Tableau. The objective is to identify food consumption patterns, nutritional awareness, and lifestyle factors that influence students' daily food choices. The interactive dashboards provide meaningful insights that can help promote healthier eating habits among students.

---

## 🎯 Objectives

- Analyze the food preferences of college students.
- Study the relationship between dietary habits and lifestyle.
- Identify common factors influencing food choices.
- Visualize trends using interactive Tableau dashboards.
- Provide insights that support healthy dietary decisions.

---

## 📊 Tools & Technologies

- Tableau Desktop
- Microsoft Excel (Dataset)
- Git & GitHub

---

## 📁 Dataset

The dataset contains information related to students' eating habits, food preferences, nutritional choices, and lifestyle patterns. It is used to create interactive dashboards for better understanding and analysis.

---

## 📈 Dashboard Features

- Food Preference Distribution
- Meal Consumption Patterns
- Healthy vs. Unhealthy Food Choices
- Lifestyle and Dietary Trends
- Interactive Filters and Visualizations
- Summary Dashboard with Key Insights

---

## 🔍 Key Insights

- Students with busy schedules tend to choose fast food more frequently.
- Home-cooked meals are generally associated with healthier eating habits.
- Nutritional awareness varies across different student groups.
- Lifestyle and daily routines have a significant impact on food choices.

---

## 🚀 How to Use

1. Clone or download this repository.
2. Open the Tableau workbook (.twb or .twbx).
3. Load the dataset if required.
4. Explore the interactive dashboards using filters and charts.
